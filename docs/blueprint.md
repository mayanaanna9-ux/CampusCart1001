# **App Name**: Campus Cart

## Core Features:

- Gmail Authentication: Allow students to sign up and log in using their Gmail accounts.
- Item Posting: Enable sellers to post items for sale with details like name, description, category, price, and condition.
- Image Upload: Allow students to upload images of their items from files (desktop) or device gallery (mobile).
- Direct Messaging: Enable buyers and sellers to communicate directly via in-app messaging.
- Profile Picture Customization: Allow students to upload profile pictures from files or choose from pre-designed avatars. Students must create profiles before anything else.
- Browse, Buy, and Sell: Enable students to browse, buy, and sell items with an intuitive interface and category sorting.
- New Post Recommendations: Suggest products the user may like by using a tool that reviews items other students post for sale.

## Style Guidelines:

- Primary color: A warm orange (#FF7B54) to convey energy and enthusiasm.
- Background color: A light, desaturated orange (#FAE8E2) provides a soft, inviting backdrop.
- Accent color: A vibrant yellow (#F0E68C) to draw the eye and highlight important information.
- Font pairing: 'Poppins' (sans-serif) for headlines, providing a modern, geometric feel, and 'PT Sans' (sans-serif) for body text, for clear readability.
- Use simple, clean icons to represent item categories and actions, ensuring ease of navigation.
- Home screen should showcase the latest items posted, with clear product images and concise descriptions.
- Use subtle animations to provide feedback and enhance the user experience, such as smooth transitions and loading indicators.
